A super simple jQuery & PHP Time Documentation Application.
created by M.McGraw

I am working on this application for 2 reasons: 1-To get more familiar with server-side scripting 2-A quick way for me to record daily time spent on billed projects. Currently this application will create a text document from user input containing username, date, start/finish times, daily hour total and any notes for that specific date. The text doc is saved on the server. The document is retrieved by submitting user name in the 'View Your Details' text box (the document is created and saved as userName.txt). The code is fully commented - free to use without a license. This project is ongoing and functionality will be added soon. Im recording project notes on the wiki @ gitHub
